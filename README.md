# Referendum2020 Аномалии общероссийского голосования по поправкам к Конституции России.
Общероссийское голосование по вопросу одобрения изменений, вносимых в Конституцию Российской Федерации, проводилось с 25 июня по 1 июля 2020 года. Последний — основной — день голосования (среда) был объявлен выходным. Жители Москвы и Нижегородской области с 25 по 30 июня по желанию могли голосовать электронно
[wiki](https://ru.wikipedia.org/wiki/%D0%9E%D0%B1%D1%89%D0%B5%D1%80%D0%BE%D1%81%D1%81%D0%B8%D0%B9%D1%81%D0%BA%D0%BE%D0%B5_%D0%B3%D0%BE%D0%BB%D0%BE%D1%81%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5_%D0%BF%D0%BE_%D0%BF%D0%BE%D0%BF%D1%80%D0%B0%D0%B2%D0%BA%D0%B0%D0%BC_%D0%BA_%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%86%D0%B8%D0%B8_%D0%A0%D0%BE%D1%81%D1%81%D0%B8%D0%B8).

В данном colab'е приводится анализ результатов с целью выявления случаев голосования с нулевой дисперсией, то есть результатов когда все УИКи внутри одного ТИКа голосуют одинаково или с минимальным разбросом. Данные случаи не имеют естественного объяснения кроме того, что подсчет голосов не осуществлялся и результат голосования был просто выдуман.

##Источники данных:
Сведения о результатах голосования предоставляет ЦИК РФ. Разнообразные цифры с разбивкой по округам доступны на странице [ЦИК](http://www.vybory.izbirkom.ru/region/izbirkom?action=show&root_a=12000007&vrn=100100163596966&region=0&global=true&type=0&prver=0&pronetvd=null). К сожалению, эти данные сильно фрагментированы и мало приспособлены для машинного анализа. 

Данные результатов и явки взяты из телеграм канала Сергея Шпилькина [RUElectionData ](https://t.me/RUElectionData).

##Основной результат:
1. Найдены сотни ТИКов имеющих близкий к нулю разброс явки и результатов голосования.

2. Найдены ТИКи, где волатильность приращения внутридневной явки между контрольными точками 10.00 12.00 15.00 и 18.00 крайне мала.

3. Данные случаи не имеют естественного объяснения кроме того, что подсчет голосов не осуществлялся.
